\doxysubsection{Detailed Description}
Class for loading and manipulating character fonts. 

Fonts can be loaded from a file, from memory or from a custom stream, and supports the most common types of fonts. See the load\+From\+File function for the complete list of supported formats.

Once it is loaded, a \doxylink{a01697}{sf\+::\+Font} instance provides three types of information about the font\+: \begin{DoxyItemize}
\item Global metrics, such as the line spacing \item Per-\/glyph metrics, such as bounding box or kerning \item Pixel representation of glyphs\end{DoxyItemize}
Fonts alone are not very useful\+: they hold the font data but cannot make anything useful of it. To do so you need to use the \doxylink{a01781}{sf\+::\+Text} class, which is able to properly output text with several options such as character size, style, color, position, rotation, etc. This separation allows more flexibility and better performances\+: indeed a \doxylink{a01697}{sf\+::\+Font} is a heavy resource, and any operation on it is slow (often too slow for real-\/time applications). On the other side, a \doxylink{a01781}{sf\+::\+Text} is a lightweight object which can combine the glyphs data and metrics of a \doxylink{a01697}{sf\+::\+Font} to display any text on a render target. Note that it is also possible to bind several \doxylink{a01781}{sf\+::\+Text} instances to the same \doxylink{a01697}{sf\+::\+Font}.

It is important to note that the \doxylink{a01781}{sf\+::\+Text} instance doesn\textquotesingle{}t copy the font that it uses, it only keeps a reference to it. Thus, a \doxylink{a01697}{sf\+::\+Font} must not be destructed while it is used by a \doxylink{a01781}{sf\+::\+Text} (i.\+e. never write a function that uses a local \doxylink{a01697}{sf\+::\+Font} instance for creating a text).

Apart from loading font files, and passing them to instances of \doxylink{a01781}{sf\+::\+Text}, you should normally not have to deal directly with this class. However, it may be useful to access the font metrics or rasterized glyphs for advanced usage.

Note that if the font is a bitmap font, it is not scalable, thus not all requested sizes will be available to use. This needs to be taken into consideration when using \doxylink{a01781}{sf\+::\+Text}. If you need to display text of a certain size, make sure the corresponding bitmap font that supports that size is used.
