\doxysubsubsection{\texorpdfstring{encode}{encode}}
{\footnotesize\ttfamily bool cpr\+::\+Cookies\+::encode \{true\}}



Should we URL-\/encode cookies when making a request. 

Based on RFC6265, it is recommended but not mandatory to encode cookies.

\DoxyHorRuler{0}
 To maximize compatibility with user agents, servers that wish to store arbitrary data in a cookie-\/value SHOULD encode that data, for \hypertarget{a00721_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{example, using Base64 \mbox{[}\+RFC4648\mbox{]}.}{example, using Base64 [RFC4648].}}\label{a00721_autotoc_md1}
