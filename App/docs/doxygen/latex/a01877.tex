\doxysubsection{Detailed Description}
Implementation of input stream based on a file. 

This class is a specialization of \doxylink{a01881}{Input\+Stream} that reads from a file on disk.

It wraps a file in the common \doxylink{a01881}{Input\+Stream} interface and therefore allows to use generic classes or functions that accept such a stream, with a file on disk as the data source.

In addition to the virtual functions inherited from \doxylink{a01881}{Input\+Stream}, \doxylink{a01877}{File\+Input\+Stream} adds a function to specify the file to open.

SFML resource classes can usually be loaded directly from a filename, so this class shouldn\textquotesingle{}t be useful to you unless you create your own algorithms that operate on an \doxylink{a01881}{Input\+Stream}.
