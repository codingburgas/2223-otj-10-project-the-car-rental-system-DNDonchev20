the JSON type enumeration 

This enumeration collects the different JSON types. It is internally used to distinguish the stored values, and the functions \doxylink{a01541_aedc7afad96292b5ab61a2e0ad3067f5f}{basic\+\_\+json\+::is\+\_\+null()}, \doxylink{a01541_a57e8411a770a6263d6d8f2116c37f3aa}{basic\+\_\+json\+::is\+\_\+object()}, \doxylink{a01541_ab5b70d60a636b9c5e10f6c8caac60b9e}{basic\+\_\+json\+::is\+\_\+array()}, \doxylink{a01541_ab303d17366c26fca12242c7f8def1bb7}{basic\+\_\+json\+::is\+\_\+string()}, \doxylink{a01541_a911b11e855e685fa59ea1d111490b36b}{basic\+\_\+json\+::is\+\_\+boolean()}, \doxylink{a01541_abd47ac8eba63833152795280f75b5851}{basic\+\_\+json\+::is\+\_\+number()} (with \doxylink{a01541_ac4b4acf2c0ad075c0dc125a65c102362}{basic\+\_\+json\+::is\+\_\+number\+\_\+integer()}, \doxylink{a01541_a5493f2ed1e07b0ece428bd5a47e2fb95}{basic\+\_\+json\+::is\+\_\+number\+\_\+unsigned()}, and \doxylink{a01541_a116cdb9300b56519fc9cf756609296cb}{basic\+\_\+json\+::is\+\_\+number\+\_\+float()}), \doxylink{a01541_aecaaa0613d3f3a5b49b34b02afc5f85d}{basic\+\_\+json\+::is\+\_\+discarded()}, \doxylink{a01541_a548d2d4013da24e7d7510d90febc80c4}{basic\+\_\+json\+::is\+\_\+primitive()}, and \doxylink{a01541_a4e05a7d5deec758f1d830741b68b4249}{basic\+\_\+json\+::is\+\_\+structured()} rely on it.

\begin{DoxyNote}{Note}
There are three enumeration entries (number\+\_\+integer, number\+\_\+unsigned, and number\+\_\+float), because the library distinguishes these three types for numbers\+: \doxylink{a01541_ae09af9c23351b7245d9be4d1b2035fef}{basic\+\_\+json\+::number\+\_\+unsigned\+\_\+t} is used for unsigned integers, \doxylink{a01541_a11e390944da90db83089eb2426a749d3}{basic\+\_\+json\+::number\+\_\+integer\+\_\+t} is used for signed integers, and \doxylink{a01541_a5b8abaebd922d82d69756327c0c347e6}{basic\+\_\+json\+::number\+\_\+float\+\_\+t} is used for floating-\/point numbers or to approximate integers which do not fit in the limits of their respective type.
\end{DoxyNote}
