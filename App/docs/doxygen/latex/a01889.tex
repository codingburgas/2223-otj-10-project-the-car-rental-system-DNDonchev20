\doxysubsection{Detailed Description}
Implementation of input stream based on a memory chunk. 

This class is a specialization of \doxylink{a01881}{Input\+Stream} that reads from data in memory.

It wraps a memory chunk in the common \doxylink{a01881}{Input\+Stream} interface and therefore allows to use generic classes or functions that accept such a stream, with content already loaded in memory.

In addition to the virtual functions inherited from \doxylink{a01881}{Input\+Stream}, \doxylink{a01889}{Memory\+Input\+Stream} adds a function to specify the pointer and size of the data in memory.

SFML resource classes can usually be loaded directly from memory, so this class shouldn\textquotesingle{}t be useful to you unless you create your own algorithms that operate on an \doxylink{a01881}{Input\+Stream}.
