\doxysubsection{Detailed Description}
Defines a system event and its parameters. 

\doxylink{a01977}{sf\+::\+Event} holds all the informations about a system event that just happened. Events are retrieved using the \doxylink{a02069_a338e996585faf82e93069858e3b531b7}{sf\+::\+Window\+::poll\+Event} and \doxylink{a02069_aaf02ab64fbc1d374eef3696df54137bc}{sf\+::\+Window\+::wait\+Event} functions.

A \doxylink{a01977}{sf\+::\+Event} instance contains the type of the event (mouse moved, key pressed, window closed, ...) as well as the details about this particular event. Please note that the event parameters are defined in a union, which means that only the member matching the type of the event will be properly filled; all other members will have undefined values and must not be read if the type of the event doesn\textquotesingle{}t match. For example, if you received a Key\+Pressed event, then you must read the event.\+key member, all other members such as event.\+mouse\+Move or event.\+text will have undefined values.
