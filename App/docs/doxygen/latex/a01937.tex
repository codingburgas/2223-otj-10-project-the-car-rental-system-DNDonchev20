Utility class providing generic functions for UTF conversions. 

\doxylink{a01937}{sf\+::\+Utf} is a low-\/level, generic interface for counting, iterating, encoding and decoding Unicode characters and strings. It is able to handle ANSI, wide, latin-\/1, UTF-\/8, UTF-\/16 and UTF-\/32 encodings.

sf\+::\+Utf$<$\+X$>$ functions are all static, these classes are not meant to be instantiated. All the functions are template, so that you can use any character / string type for a given encoding.
