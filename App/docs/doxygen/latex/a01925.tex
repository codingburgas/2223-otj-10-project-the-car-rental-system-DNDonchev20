\doxysubsection{Detailed Description}
Defines variables with thread-\/local storage. 

This class manipulates void\texorpdfstring{$\ast$}{*} parameters and thus is not appropriate for strongly-\/typed variables. You should rather use the \doxylink{a01929}{sf\+::\+Thread\+Local\+Ptr} template class.
