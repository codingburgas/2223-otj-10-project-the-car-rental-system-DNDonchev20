\doxysubsection{Detailed Description}
Graphical text that can be drawn to a render target. 

\doxylink{a01781}{sf\+::\+Text} is a drawable class that allows to easily display some text with custom style and color on a render target.

It inherits all the functions from \doxylink{a01793}{sf\+::\+Transformable}\+: position, rotation, scale, origin. It also adds text-\/specific properties such as the font to use, the character size, the font style (bold, italic, underlined and strike through), the text color, the outline thickness, the outline color, the character spacing, the line spacing and the text to display of course. It also provides convenience functions to calculate the graphical size of the text, or to get the global position of a given character.

\doxylink{a01781}{sf\+::\+Text} works in combination with the \doxylink{a01697}{sf\+::\+Font} class, which loads and provides the glyphs (visual characters) of a given font.

The separation of \doxylink{a01697}{sf\+::\+Font} and \doxylink{a01781}{sf\+::\+Text} allows more flexibility and better performances\+: indeed a \doxylink{a01697}{sf\+::\+Font} is a heavy resource, and any operation on it is slow (often too slow for real-\/time applications). On the other side, a \doxylink{a01781}{sf\+::\+Text} is a lightweight object which can combine the glyphs data and metrics of a \doxylink{a01697}{sf\+::\+Font} to display any text on a render target.

It is important to note that the \doxylink{a01781}{sf\+::\+Text} instance doesn\textquotesingle{}t copy the font that it uses, it only keeps a reference to it. Thus, a \doxylink{a01697}{sf\+::\+Font} must not be destructed while it is used by a \doxylink{a01781}{sf\+::\+Text} (i.\+e. never write a function that uses a local \doxylink{a01697}{sf\+::\+Font} instance for creating a text).

See also the note on coordinates and undistorted rendering in \doxylink{a01793}{sf\+::\+Transformable}.
