struct nlohmann\+::json\+\_\+sax$<$ Basic\+Json\+Type $>$}
SAX interface. 

This class describes the SAX interface used by \doxylink{a01541_a12b382c6407da5543827ce4b24bb5008}{nlohmann\+::json\+::sax\+\_\+parse}. Each function is called in different situations while the input is parsed. The boolean return value informs the parser whether to continue processing the input.
