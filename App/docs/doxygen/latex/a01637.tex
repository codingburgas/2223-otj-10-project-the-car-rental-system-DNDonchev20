\doxysubsection{Detailed Description}
Manages and instantiates sound file readers and writers. 

This class is where all the sound file readers and writers are registered. You should normally only need to use its registration and unregistration functions; readers/writers creation and manipulation are wrapped into the higher-\/level classes \doxylink{a01605}{sf\+::\+Input\+Sound\+File} and \doxylink{a01621}{sf\+::\+Output\+Sound\+File}.

To register a new reader (writer) use the \doxylink{a01637_acaf1f0bf1b7c7b9988ec76b33e33fe95}{sf\+::\+Sound\+File\+Factory\+::register\+Reader} (register\+Writer) static function. You don\textquotesingle{}t have to call the unregister\+Reader (unregister\+Writer) function, unless you want to unregister a format before your application ends (typically, when a plugin is unloaded).

Usage example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{sf::SoundFileFactory::registerReader<MySoundFileReader>();}
\DoxyCodeLine{sf::SoundFileFactory::registerWriter<MySoundFileWriter>();}

\end{DoxyCode}
