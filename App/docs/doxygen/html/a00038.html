◆ CPR_VERSION_NUM

#define CPR_VERSION_NUM   0x10702

CPR version as a single hex digit.

it can be split up into three parts: 0xAABBCC AA: The current CPR major version number in a hex format. BB: The current CPR minor version number in a hex format. CC: The current CPR patch version number in a hex format.

Examples: '0x010702' -> 01.07.02 -> CPR_VERSION: 1.7.2 '0xA13722' -> A1.37.22 -> CPR_VERSION: 161.55.34
