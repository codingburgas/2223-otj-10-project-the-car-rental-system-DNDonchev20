\doxysubsection{Detailed Description}
Abstract base class for objects that can be drawn to a render target. 

\doxylink{a01693}{sf\+::\+Drawable} is a very simple base class that allows objects of derived classes to be drawn to a \doxylink{a01749}{sf\+::\+Render\+Target}.

All you have to do in your derived class is to override the draw virtual function.

Note that inheriting from \doxylink{a01693}{sf\+::\+Drawable} is not mandatory, but it allows this nice syntax \"{}window.\+draw(object)\"{} rather than \"{}object.\+draw(window)\"{}, which is more consistent with other SFML classes.
